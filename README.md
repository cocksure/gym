# Gym Tracker - Приложение для учета тренировок

Веб-приложение для учета тренировок с поддержкой упражнений с GIF-анимацией.

## Функционал

- Просмотр упражнений по категориям с GIF-анимацией
- Создание тренировок с выбором категории
- Запись подходов, повторений и веса для каждого упражнения
- Админка для управления упражнениями и категориями
- Импорт упражнений из ExerciseDB API

## Установка и запуск

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Создание суперпользователя

```bash
python manage.py createsuperuser
```

### 3. Импорт тестовых данных

```bash
python manage.py import_exercises
```

Эта команда создаст:
- 5 категорий упражнений (Грудь, Плечи, Спина, Ноги, Руки)
- 3 примера упражнений с GIF-анимацией

### 4. Запуск сервера

```bash
python manage.py runserver
```

Приложение будет доступно по адресу: http://127.0.0.1:8000/

## Использование

### Как это работает:

**1. Один раз добавляешь упражнения через админку**
- Заходишь в админку: http://127.0.0.1:8000/admin/
- Добавляешь все упражнения с GIF
- Упражнения группируются по категориям (Грудь, Спина, Ноги и т.д.)

**2. Каждую тренировку записываешь через UI**
- Создаешь новую тренировку (например "Грудь + Бицепс")
- Выбираешь упражнения из списка с GIF-превью
- Записываешь: подходы, повторения, вес
- В следующий раз смотришь GIF и вспоминаешь что делал

**Пример:**
- **Понедельник:** Создаешь тренировку "Грудь + Бицепс"
  - Жим лёжа: 3 подхода × 10 повторений × 60 кг
  - Разводка на тренажере: 3 × 12 × 20 кг (GIF показывает как делать)
  - Подъём на бицепс: 3 × 12 × 15 кг

- **Среда:** Создаешь тренировку "Спина + Трицепс"
  - Подтягивания: 3 × 8 × свой вес
  - Тяга штанги: 3 × 10 × 50 кг
  - Французский жим: 3 × 12 × 20 кг

### Навигация:

1. **Главная** (/) - Просмотр всех упражнений по категориям
2. **Мои тренировки** (/workouts/) - История всех тренировок с GIF
3. **Новая тренировка** (/workouts/create/) - Начать новую
4. **Админка** (/admin/) - Добавить новые упражнения

### Для администратора

Админка доступна по адресу: http://127.0.0.1:8000/admin/

В админке можно:
- Управлять категориями упражнений
- Добавлять/редактировать упражнения
- Просматривать все тренировки пользователей

### Способ 2: Через ExerciseDB API

**Вариант А: Бесплатный API** (есть лимиты)
- https://exercisedb.dev
- Документация: https://docs.exercisedb.dev
- Не рекомендуется для продакшена

**Вариант Б: RapidAPI** (платный, стабильный)
1. Зарегистрируйтесь: https://rapidapi.com/justin-WFnsXH_t6/api/exercisedb
2. Получите API ключ
3. Откройте `exercises/management/commands/import_exercises.py`
4. Поставьте `use_rapidapi = True`
5. Вставьте ваш ключ в `YOUR_API_KEY_HERE`
6. Запустите:

```bash
python manage.py import_exercises --limit 100
```

## Модели данных

### ExerciseCategory
- name - название категории
- slug - URL-совместимое имя

### Exercise
- name - название упражнения
- category - категория (FK)
- gif_url - ссылка на GIF-анимацию
- equipment - необходимое оборудование

### WorkoutDay
- date - дата тренировки (автоматически)
- name - название тренировки (например: "Грудь + Бицепс")
- notes - заметки о тренировке

### WorkoutExercise
- workout_day - день тренировки (FK)
- exercise - упражнение (FK)
- sets - количество подходов
- reps - количество повторений
- weight - вес (кг)

## Технологии
